# 简介

这是一个使用[Milkio](https://github.com/akirarika/milkio)和Electron制作的bilibili客户端

*ps: 使用下载视频功能需要在config.json文件中填写ffmpeg.exe的绝对路径*

# 启动
```bash
co dev
```
